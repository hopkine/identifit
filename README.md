# Identifit 👗

A personal style companion that helps you **capture your identity in every fit**. [page:0]  

[👉 Try Live Demo](https://identifit.vercel.app/demo-viewer.html) · [✨ View Features](https://identifit.vercel.app/#features) [page:0]

## What it does

Identifit is a fashion and style tracking app that helps you track outfits, discover trends, and get smarter outfit suggestions over time. [page:0]  
It’s built as a portfolio project to showcase end‑to‑end product thinking, from UX to implementation and deployment. [page:0]

## Features

- **OOTD Tracking (📸)**  
  Upload your daily outfits to build a visual diary of your style, track streaks, and see how much of your wardrobe you actually wear. [page:0]

- **Personalized Suggestions (✨)**  
  Get outfit recommendations based on your preferences, the weather, upcoming occasions, and the pieces you already own. [page:0]

- **Recreate Inspirations (🔄)**  
  Find similar items in your closet so you can recreate looks from social media and fashion magazines instead of always buying new pieces. [page:0]

- **Monthly Style Recap (📊)**  
  Review your style journey with monthly recaps showing most‑worn pieces, color preferences, and how your style evolves over time. [page:0]

## Getting started (local development)

> Note: Adjust commands to match your actual repo name and scripts.

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/identifit.git
   cd identifit
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the app**

   ```bash
   npx expo start
   ```

4. **Open on device or web**

   - Scan the QR code with the Expo Go app (iOS/Android), or  
   - Press `w` in the Expo CLI to open the web version in your browser.

If you use Supabase or other APIs, create a `.env` file and add your environment variables, then ensure they’re loaded via Expo config.
